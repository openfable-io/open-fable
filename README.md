# OpenFable

**Verified output. Open to everyone.**

AI output should be trustworthy by default. Not because a corporation guarantees it - because the process itself is verifiable.

OpenFable is an open-source project exploring verified AI output through silent critique loops. No enterprise contract. No gatekeeping.

## Status

Early stage. Building in the open.

## Ecosystem

Part of the DAI (Decentralized Autonomous Inference) ecosystem.

Built on the work of [KingLabsA](https://github.com/KingLabsA) and the FableForge community.
